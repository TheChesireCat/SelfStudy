## üìä Organization

Books are organized hierarchically:
- **01-Foundations**: Core mathematics and computer science fundamentals
- **02-Network-Science**: Graph theory, complex networks, and machine learning on graphs
- **03-Complex-Systems**: Nonlinear dynamics, chaos theory, and urban science
- **04-Applied-Methods**: Epidemiology and research design methods
- **05-General-Reading**: Popular science and broader perspectives

Each category contains PDFs, study notes, code experiments, and related materials.
